# Gemma 4 - Google's Open Weights Model Family

[![Vue 3](https://img.shields.io/badge/Vue-3.5-blue?style=flat-square)](https://vuejs.org/)
[![Vite](https://img.shields.io/badge/Vite-8.0.0-green?style=flat-square)](https://vitejs.dev/)
[![MIT](https://img.shields.io/badge/License-MIT-yellow?style=flat-square)](https://opensource.org/licenses/MIT)

This is a simple multilingual website (English, Chinese, Japanese) introducing the Gemma 4 open weights model family released by Google.

## Getting Started

```bash
npm install
npm run dev
```

The app will open at http://localhost:5173.

## Pages

- **Home** – Hero banner, language loader, model specification cards, blog previews, and sponsor logotype collection.
- **Models** – A specification grid showing Gemma 4, Gemma 4p, Gemma 4s, Gemma 4c, Gemma 4m, and Gemma 4h.
- **About** – Related links, experimental models, and related repositories.
- **Sponsors** – List of project sponsors and backers.
- **Blog** – Latest posts and categories (posts are still being added).

## Tech Stack

- **Vue 3** – Composition API, component-based UI.
- **Vite** – Fast dev server + build tool.
- **Vue Router** – SPA routing with multiple page transitions.
- **ES Modules** – Pure JS with no build tools beyond Vite.
